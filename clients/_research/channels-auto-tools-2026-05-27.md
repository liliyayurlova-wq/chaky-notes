## 💡 Готовая автоматизация под каждый канал

### 🔥 TG-посевы (наш основной продукт)
```
1. VK API → поиск каналов по нише (авто)
2. CSV-база → фильтр по подписчикам/цене (авто через tg-channel-scout.py)
3. ChatGPT → текст поста для селлера (авто)
4. TGStat → верификация канала (ручная проверка)
5. Оплата админу → пост выходит (ручная)
```
**Автоматизация:** 80% (шаги 1-3, 5 вручную)
**Инструменты:** VK API + DeepSeek/OpenRouter + tg-channel-scout.py

### 📺 YouTube Shorts (для личного бренда селлера)
```
1. YouTube API → сбор трендов в нише (авто)
2. ChatGPT → сценарий 60 сек (авто)
3. ElevenLabs → озвучка (авто)
4. CapCut → монтаж (ручной/полуавто)
5. YouTube → публикация (ручная)
```
**Автоматизация:** 50%
**Инструменты:** YouTube API + ChatGPT + ElevenLabs + CapCut

### 📝 Статьи на VC/Пикабу (SEO + трафик)
```
1. DeepSeek → анализ конкурентов и болей ЦА (авто)
2. ChatGPT → текст статьи 3000+ знаков (авто)
3. Kandinsky → 1-2 иллюстрации (авто)
4. VC/Пикабу → публикация (ручная)
```
**Автоматизация:** 75%
**Инструменты:** DeepSeek + ChatGPT + Kandinsky

### 🎯 Яндекс.Директ (платный трафик, не входит в наши услуги)
```
1. YandexGPT → подбор ключевых слов (авто)
2. ChatGPT → тексты объявлений (авто)
3. Yandex Direct API → загрузка кампании (авто)
4. Мониторинг → ручная оптимизация
```
**Автоматизация:** 70%
**Инструменты:** YandexGPT + ChatGPT + Yandex Direct API
